Fordel i KIWI: 15 % Trumf-bonus på frukt og grønt
KIWI PLUSS-kunder sparer alltid 15 % Trumf-bonus på all fersk frukt og grønt hos KIWI, og på Trippel-Trumf-torsdager sparer KIWI PLUSS-kundene hele 17 % prosent Trumf-bonus på all fersk frukt og grønt.
KIWIs bleieavtale

Bind- og tampongavtalen
Alle KIWI PLUSS-kunder sparer automatisk 50 prosent Trumf-bonus på hver eneste pakke med bind, tamponger og truseinnlegg hos KIWI.
Alt du trenger å vite om KIWI PLUSS
Som KIWI PLUSS-kunde får du alltid 1 % Trumf-bonus på dagligvarer, 15 % på all frukt og grønt, og 50 % Trumf-bonus på alle bleier, bind, tamponger og truseinnlegg. På Trippel-Trumf-torsdag får du 17 % bonus på all fersk frukt og grønt, og 3 % bonus på andre dagligvarer. KIWI PLUSS er gratis og helt uforpliktende.
